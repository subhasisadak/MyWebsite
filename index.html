QUALIFICATION

Education

I am delighted to share a glimpse of my educational journey, which has been instrumental in shaping my expertise in Electrical Engineering.

In 2022, I graduated with distinction, earning a Bachelor's degree in Electrical Engineering from Techno Main Salt Lake College. This esteemed institution is affiliated with Maulana Abul Kalam Azad University of Technology. During my B.Tech journey, I dedicated myself to academic excellence and achieved an outstanding Cumulative Grade Point Average (CGPA) of 8.8. This accomplishment reflects not only my academic prowess but also my commitment to mastering the intricacies of Electrical Engineering.

My educational pursuit also includes a Diploma in Electrical Engineering, which I successfully completed in 2019 at JLD College of Engineering and Management. This diploma, accredited by the West Bengal State Council of Technical & Vocational Education and Skill Development, laid a solid foundation for my technical skills and knowledge. I am proud to have graduated with a commendable 81.9% marks, showcasing my consistent dedication and proficiency in the core principles of Electrical Engineering.

Skills

Welcome to the skills section of my website, where I'm excited to showcase the diverse set of technical skills I've acquired to excel in the world of software development.

Frontend Development: In the realm of Frontend development, I have honed my skills in HTML5, CSS, and JavaScript, leveraging these foundational technologies to create intuitive and visually appealing user interfaces. Furthermore, I am proficient in React, a powerful JavaScript library for building interactive and dynamic user interfaces, and I apply Bootstrap to enhance the responsiveness and styling of Web Applications.

Backend Development: For Backend development, I've cultivated expertise in Python, as well as Core and Advanced Java. This enables me to develop robust server-side logic and efficiently handle data processing. Additionally, I specialize in building Rest APIs and microservices using the Java framework Spring Boot 3, ensuring seamless communication between the Frontend and Backend components of applications.

Database Design: I possess a solid understanding of Database Management Systems (DBMS) and Query Languages, particularly MySQL and SQL Server. This knowledge empowers me to design efficient and scalable database structures, ensuring optimal data storage and retrieval for applications.

Data Structures and Algorithms: A deep understanding of Data Structures and Algorithms is at the core of my skill set. This proficiency allows me to devise efficient algorithms, optimize code for performance, and solve complex problems with a systematic approach. It serves as the backbone of my ability to create robust and scalable software solutions.
